TravelWise Developer API

Welcome to the TravelWise API documentation. Our REST API exposes all backend resources including travel destinations, local attractions, package bundles, and custom traveler itineraries.

Authentication Requirement
Session cookies are used for authentication. All write operations (POST, PUT, PATCH, DELETE) and user-private records (Trips) require a valid active session. Standard error formatting returns JSON with detail exceptions.

Destinations API

GET /api/destinations/

Retrieve a paginated list of all destinations. Supports dynamic filters, search queries, and custom sorting.

Query Parameters:
Parameter Type Description
searchstringSearch term matched against destination name, city, state, or category.
statestringExact state matching (case-insensitive).
categorystringExact category matching.
orderingstringSort result set. Allowed: destination_name, average_cost_per_day, average_rating. Prefix with - for descending.
Example Request:
GET /api/destinations/?category=Beach&ordering=average_cost_per_day
Example Response (Status 200 OK):
{
  "count": 1,
  "next": null,
  "previous": null,
  "results": [

      "destination_id": 1,
      "destination_name": "Goa Beach",
      "city": "Panaji",
      "state": "Goa",
      "region": "West",
      "category": "Beach",
      "description": "Lovely sunny beaches.",
      "best_season": "Winter",
      "ideal_days": 3,
      "budget_level": "Moderate",
      "average_cost_per_day": "4500.00",
      "family_friendly": true,
      "couple_friendly": true,
      "solo_friendly": true,
      "average_rating": "4.50",
      "image": "/media/destinations/goa.jpg"
    }
  ]
}
POST /api/destinations/

Create a new travel destination record. Staff Only

Required Body Parameters (JSON):
  • destination_name (string, unique)
  • city (string)
  • category (string)
  • description (string)
  • best_season (string)
  • budget_level (string: 'Budget', 'Moderate', 'Luxury')
  • average_cost_per_day (decimal)
Example Request Body:
{
  "destination_name": "Manali Hills",
  "city": "Manali",
  "state": "Himachal Pradesh",
  "category": "Hill Station",
  "description": "Snowy valley retreat.",
  "best_season": "Summer",
  "budget_level": "Moderate",
  "average_cost_per_day": "3500.00"
}

POST /api/trips/

Create a trip. Dynamically auto-generates structured ItineraryDays using templates or fallbacks. Authenticated Only

Required Body Parameters (JSON):
  • destination (integer, ID)
  • start_date (string, YYYY-MM-DD)
  • end_date (string, YYYY-MM-DD)
  • number_of_travelers (integer)
  • budget (decimal)
  • travel_type (string: 'Solo', 'Couple', 'Family', 'Friends')
Example Request Body:
{
  "destination": 1,
  "start_date": "2026-10-10",
  "end_date": "2026-10-12",
  "number_of_travelers": 2,
  "budget": "15000.00",
  "travel_type": "Couple"
}
Example Response (Status 201 Created):
{

  "user_username": "traveller1",
  "start_date": "2026-10-10",
