About TravelWise

TravelWise is an intuitive, collaborative project dashboard designed to help travelers discover destinations, set budgets, and compile comprehensive itineraries in one place.

Developed as a modern web app, this tool leverages Django's robust backend architecture to power account administration, secure API synchronization, and itinerary scheduling modules.
